# Minnytime: Tabletop Snooker

A mobile-first local two-player snooker game for the browser.

## Current build

- Two-player pass-and-play
- Player names and live scoreboard
- Tap or drag to aim
- Fine aim buttons
- Adjustable shot power
- Ball-to-ball collision physics
- Cushion rebounds, friction and six pockets
- Reds followed by colours
- Final yellow-to-black clearance
- Legal pots keep the turn
- Misses and fouls pass the turn
- Fouls award at least four points
- Cue-ball replacement after a scratch
- Pause, restart, rules and rematch screens

## Controls

1. Tap or drag anywhere on the table to aim.
2. Set the shot power.
3. Press **Take Shot**.
4. Pass the device to the other player when the turn changes.
